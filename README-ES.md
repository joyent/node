Motor de E/S Orientado al Evento para la Maquina V8 de JavaScript.
===

Para compilar:

    ./configure
    make
    make install

Para ejecutar los tests:

    make test

Para construir la documentación:

    make doc

Para leer la documentación:

    man doc/node.1
